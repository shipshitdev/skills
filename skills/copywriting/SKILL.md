# Copywriting

Produce clear, credible, and action-oriented marketing copy aligned with user intent and business goals.

You may **not** fabricate claims, statistics, testimonials, or guarantees.

## When to Use

- Writing homepage, landing page, pricing, or feature page copy
- Creating CTAs, headlines, and value propositions
- Improving conversion rates on existing pages

## Phase 1 — Context Gathering (Mandatory)

Gather or confirm before writing any copy. Ask if missing.

**Page Purpose**

- Page type (homepage, landing, pricing, feature, about)
- ONE primary CTA
- Secondary action if any

**Audience**

- Target customer or role
- Primary problem they are solving
- What they have already tried
- Main objections or hesitations
- Language they use to describe the problem

**Product / Offer**

- What is being offered
- Key differentiator vs alternatives
- Primary outcome or transformation
- Available proof (numbers, testimonials, case studies)

**Context**

- Traffic source (ads, organic, email, referrals)
- Awareness level (unaware / problem-aware / solution-aware / product-aware)

## Phase 2 — Copy Brief Lock (Hard Gate)

Present a **Copy Brief Summary** (4-6 bullets) and ask:

> "Does this copy brief accurately reflect what we're trying to achieve? Please confirm or correct anything before I write copy."

**Do NOT write copy until confirmation is given.**

## Phase 3 — Copywriting Principles

**Core (non-negotiable):**

- Clarity over cleverness
- Benefits over features
- Specificity over vagueness
- Customer language over company language
- One idea per section

**Always connect:** Feature → Benefit → Outcome

**Style:**

- Simple over complex
- Active over passive
- Confident over hedged
- Show outcomes, not adjectives
- Avoid buzzwords

**Claim Discipline:**

- No fabricated data or testimonials
- No implied guarantees
- Mark placeholders clearly when proof is missing

## Phase 4 — Page Structure

### Above the Fold

- **Headline:** Single most important message, outcome-focused, 50-60 chars
- **Subheadline:** Adds clarity or context, 1-2 sentences max
- **Primary CTA:** Action-oriented, describes what user gets

### Core Sections (Use as Appropriate)

1. Social proof (logos, stats, testimonials)
2. Problem / pain articulation
3. Solution + key benefits (3-5 max)
4. How it works (3-4 steps)
5. Objection handling (FAQ, comparisons, guarantees)
6. Final CTA with recap and risk reduction

## Phase 5 — Output Format

For each copy request, provide:

- **Page Copy** organized by section with clear labels
- **Alternatives** — 2-3 options for headlines and primary CTAs, each with brief rationale
- **Annotations** — for key sections, explain why this copy was chosen

## Completion Criteria

This skill is complete ONLY when:

- Copy brief has been confirmed
- Page copy delivered in structured form
- Headline and CTA alternatives provided
- Assumptions documented

## Key Principle

Good copy does not persuade everyone. It persuades **the right person** to take **the right action**.

If the copy feels clever but unclear, rewrite it until it feels obvious.

## Limitations

- Use only when task matches marketing copy scope.
- Never fabricate proof or create misleading claims.
