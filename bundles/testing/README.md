# Ship Shit Dev - Testing Bundle

Testing, QA, and CI/CD automation skills

## Installation

```bash
/plugin marketplace add shipshitdev/skills
/plugin install shipshitdev-testing@shipshitdev
```

## Included Skills

- `playwright-e2e-init`
- `testing-expert`
- `testing-cicd-init`
- `nestjs-testing-expert`
- `qa-reviewer`
- `husky-test-coverage`
