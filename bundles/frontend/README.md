# Ship Shit Dev - Frontend Bundle

Frontend development and design skills

## Installation

```bash
/plugin marketplace add shipshitdev/skills
/plugin install shipshitdev-frontend@shipshitdev
```

- `component-library`
- `accessibility`
- `design-consistency-auditor`
